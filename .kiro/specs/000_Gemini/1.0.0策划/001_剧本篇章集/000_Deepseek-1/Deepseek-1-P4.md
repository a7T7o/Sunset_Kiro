# 篇章四：地牢探索详细策划

## 一、地牢地图设计

### 1.1 整体结构

地牢位于灰碑谷村后山的“遗忘裂隙”中，共**六层**，每层面积逐层增大（第一层最小，第六层最大）。每层包含以下房间类型：

| 房间类型 | 描述 | 出现层数 | 特殊机制 |
|----------|------|----------|----------|
| 入口房间 | 每层起点，安全区（无怪物），设有存档点（可手动保存） | 1-6 | 可放置逃脱绳（使用后返回入口） |
| 战斗房间 | 普通怪物刷新区，数量2-4组 | 1-6 | 怪物会巡逻、警戒 |
| 宝箱房间 | 包含1-3个上锁宝箱，需钥匙开启 | 2-6 | 宝箱材质决定解锁概率 |
| 精英房间 | 包含1个精英怪物，击败后掉落稀有钥匙或图纸 | 3-6 | 精英怪强度≈同层普通怪2倍 |
| 陷阱房间 | 地板有尖刺/落石，触碰损失生命 | 4-6 | 可通过观察地面纹理规避 |
| 记忆裂隙 | 特殊房间，触发记忆闪回剧情 | 3,5,6 | 与主线相关 |
| Boss房间 | 每层最终区域，击败Boss才能进入下一层 | 1-6 | 需消耗Boss钥匙（特殊物品） |

### 1.2 每层详细参数

| 层数 | 主题 | 推荐工具等级 | 大小（房间数） | 怪物种类 | 特殊产出 |
|------|------|--------------|----------------|----------|----------|
| 1 | 荒废矿道 | 木/石 | 6-8 | 史莱姆、骷髅士兵 | 铜矿石、木钥匙 |
| 2 | 骷髅墓穴 | 铁 | 8-10 | 骷髅弓箭手、骷髅法师 | 铁矿石、银钥匙 |
| 3 | 兽人营地 | 铁/钢 | 10-12 | 兽人战士、兽人萨满 | 钢矿石、金钥匙、记忆碎片 |
| 4 | 时间回廊 | 钢 | 12-14 | 时间畸变体（新）、记忆水母 | 水晶、自动灌溉核心图纸 |
| 5 | 记忆殿堂 | 钢/金 | 14-16 | 记忆守卫（镜像）、精英兽人 | 完整记忆核心、传说武器图纸 |
| 6 | 裂隙核心 | 金/传说 | 16-18 | 时间吞噬者分身、最终Boss | 时空锚点、结局道具 |

```json
{
  "dropConfigs": [
    { "item": "SlimeGoo", "minAmount": 1, "maxAmount": 2, "dropChance": 1.0, "quality": "Normal" },
    { "item": "CopperOre", "minAmount": 1, "maxAmount": 1, "dropChance": 0.2, "quality": "Normal" }
  ],
  "spreadRadius": 0.5,
  "guaranteeOneDrop": false
}
```

## 三、战斗系统设计（基于现有框架扩展）

### 3.1 现有基础

- `PlayerToolHitEmitter` 负责检测玩家工具/武器是否命中目标，发送 `ToolHitContext`。
- 目标需要实现 `IHittable` 接口（需新增），包含 `OnHit(ToolHitContext context)` 方法。

### 3.2 需要新增的组件

1. **`IHittable` 接口**：定义被击打时的行为。
2. **`EnemyController`**：实现 `IHittable`，管理敌人AI、状态、动画。
3. **`PlayerCombatController`**（可选）：整合武器切换、攻击冷却、连招等。
4. **`HealthSystem`**：管理生命值、死亡事件。
5. **`DamageCalculator`**：静态类，计算最终伤害（基于武器攻击力、敌人防御、暴击、技能加成）。

### 3.3 伤害计算公式

最终伤害 = (武器基础攻击力 × 武器品质倍率 + 玩家力量加成) × 技能倍率 × 暴击倍率 - 敌人防御（但防御减伤非线性，例如：伤害 = 攻击力² / (攻击力 + 防御) 或简单减法但最低伤害1）

为简化，采用：**伤害 = 攻击力 × (1 + 暴击修正) - 防御**，最低1点。

- 攻击力：武器 `WeaponData.attackPower` × 品质倍率（`ItemQuality` 乘数作用于攻击力，例如普通1.0，稀有1.2，史诗1.5，传说2.0）。
- 暴击：武器 `criticalChance` + 技能加成，暴击时伤害 × `criticalDamageMultiplier`。
- 防御减免：敌人防御直接减去伤害（若防御 > 伤害，则伤害=1）。

### 3.4 玩家生命与死亡

- 玩家初始生命100，可通过装备、食物永久提升。
- 受到伤害时减少生命，生命归零则触发死亡事件：掉落所有物品（死亡全掉落），在村落复活（床/医疗点）。
- 地牢内死亡：所有背包物品掉落在地牢（生成掉落包），玩家在入口复活，需回去拾取（类似暗黑风格）。

### 3.5 敌人AI简单设计

采用有限状态机：巡逻 → 索敌 → 攻击 → 追击 → 返回。

- 巡逻：在固定路径点移动。
- 索敌：检测到玩家后进入追击。
- 攻击：在攻击范围内触发攻击动画和伤害判定。
- 返回：超出追击范围后返回巡逻点。

### 3.6 武器与工具统一

工具（斧头、镐）也有 `canDealDamage` 字段，可对怪物造成伤害（但效率低），鼓励玩家制作专用武器。

## 四、Boss战设计

### 4.1 第一层Boss：巨型史莱姆

- **血量**：200
- **攻击方式**：跳跃砸地（范围伤害），分裂（半血时分裂成2个小史莱姆）
- **场地**：圆形房间，有4根柱子可躲避
- **掉落**：木钥匙×3，铜矿石×10，记忆碎片·起始（解锁锯木台Lv2）

### 4.2 第二层Boss：骷髅王

- **血量**：350
- **攻击方式**：挥剑（近战），召唤骷髅弓箭手（召唤2只），冲刺斩
- **场地**：墓穴大厅，有棺材可作掩体
- **掉落**：银钥匙×2，铁矿石×15，铁质武器图纸

### 4.3 第三层Boss：兽人酋长

- **血量**：600
- **攻击方式**：旋风斩（范围），投掷飞斧（远程），怒吼（增加自身攻击力）
- **场地**：营地广场，有篝火可点燃造成额外伤害
- **掉落**：金钥匙×1，钢矿石×20，自动进料器图纸

### 4.4 第四层Boss：时间扭曲者

- **血量**：500
- **攻击方式**：减速光环（降低玩家速度），时间炸弹（延迟爆炸），瞬移
- **场地**：回廊迷宫，需要利用机关解除减速
- **掉落**：自动灌溉核心图纸，记忆碎片·时间

### 4.5 第五层Boss：记忆守卫·真身

- **血量**：800
- **攻击方式**：复制玩家装备（根据玩家武器切换攻击模式），分身术（制造2个残影）
- **场地**：镜子大厅，可击碎镜子消除分身
- **掉落**：完整记忆核心，传说武器图纸

### 4.6 第六层Boss：时间裂隙吞噬者

- **血量**：1200（分三个阶段）
- **阶段1**：本体攻击，召唤时间畸变体小怪
- **阶段2**：半血后进入裂隙，玩家需在限时内击破三个核心
- **阶段3**：回到现实，吞噬者狂暴，攻击力翻倍
- **场地**：裂隙核心，四周漂浮记忆碎片
- **掉落**：时空锚点（结局道具），传说级材料

### 5.2 宝箱设计

宝箱使用 `StorageData` 配置，`isLockable=true`，`defaultLocked=true`。宝箱材质决定基础解锁概率（`baseUnlockChance`），与钥匙的 `unlockChance` 叠加计算最终概率。成功开锁保留钥匙，失败消耗钥匙。

| 宝箱材质 | 基础解锁概率 | 常见位置 | 奖励类型 |
|----------|--------------|----------|----------|
| Wood | 20% | 1-2层 | 基础矿石、种子、少量铜币 |
| Iron | 15% | 2-4层 | 中级矿石、武器图纸、银币 |
| Special | 10% | 3-5层 | 稀有材料、记忆碎片、金币 |
| Steel | 5% | 5-6层 | 传说图纸、完整记忆核心 |

### 5.4 背包博弈设计要点

1. **携带钥匙 vs 战利品格子**：背包只有24格（基础），每把钥匙占1格，带3把钥匙就只剩21格装战利品。玩家必须权衡：带更多钥匙去开更多宝箱，还是留格子装更多掉落物。
2. **开锁概率风险**：低概率钥匙可能浪费格子却开锁失败，促使玩家考虑是否带多把相同钥匙。
3. **逃脱绳也占格子**：逃脱绳（可返回入口）也是1格，带不带？
4. **死亡全掉落**：如果在地牢死亡，所有物品掉落（包括钥匙和战利品），增加风险。
5. **多层递进**：深层地牢钥匙更难获得，但宝箱奖励更好，引导玩家逐步挑战。

**设计检查**：玩家每次进地牢前必须做决策：带3把金钥匙去深层赌大件，还是带满食物稳扎稳打，留1格给逃脱绳。这正是铁律三“钥匙背包博弈”的核心体现。
