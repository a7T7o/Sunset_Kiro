# 篇章五：制作系统详细策划

## 一、制作系统整体框架

### 1.1 制作方式分类

基于 `CraftingService` 和 `RecipeData` 框架，制作分为三类：

| 类型 | 所需设施 | 对应 `requiredStation` | 说明 |
|------|----------|------------------------|------|
| 手工制作 | 无（玩家随身） | `None` | 基础配方，无需工作台 |
| 工作台制作 | 对应工作台 | `CookingPot` / `Furnace` / `MagicTower` / `AnvilForge` / `Workbench` / `AlchemyTable` / `Grill` | 需站在对应工作台前 |
| 特殊制作 | 特定地点/NPC | 无（通过任务解锁） | 如铁匠铺升级、村长赠礼 |

### 1.2 配方通用结构

每个 `RecipeData` 包含：
- `recipeID`：唯一ID
- `resultItemID`：产物ID
- `resultAmount`：产物数量
- `ingredients`：材料列表（`List<RecipeIngredient>`，含 itemID + amount）
- `requiredStation`：所需工作台类型
- `requiredLevel`：所需技能等级（制作/烹饪/战斗等）
- `craftingTime`：制作时间（秒）
- `unlockedByDefault`：是否默认解锁
- `isHiddenRecipe`：是否为隐藏配方（需特殊方式解锁）
- `unlockCondition`：解锁条件描述（用于任务/记忆闪回）
- `craftingExp`：获得制作经验

### 1.3 工作台三阶段演进（复用已有 WorkstationData 字段）

每个工作台通过 `WorkstationData.craftTimeMultiplier` 和升级状态机实现三阶段：

| 阶段 | 名称 | `craftTimeMultiplier` | 批量制作能力 | 自动功能 | 解锁方式 |
|------|------|------------------------|--------------|----------|----------|
| 1 | 手搓期 | 1.0 | 每次1个 | 无 | 初始可用 |
| 2 | 半自动期 | 0.7 | 可批量5个 | 无 | 记忆闪回/图纸 |
| 3 | 全自动期 | 0.5 | 可批量10个 | 自动进料/出料 | 地牢掉落图纸+材料 |

**视觉对应**：每个工作台的美术素材都有多阶段（如锯木台4级、熔炉9套、炼药台3级），可直接映射。

## 二、各工作台配方详细设计

### 2.1 手工制作（`requiredStation = None`）

手工配方主要为最基础的合成，无需工作台，适合初期。

| 配方名称 | recipeID | 材料 | 产物 | 数量 | 制作时间 | 解锁条件 | 经验 |
|----------|----------|------|------|------|----------|----------|------|
| 木棍 | 10001 | 木材×1 | 木棍×2 | 2s | 默认 | 1 |
| 火把 | 10002 | 木棍×1 + 史莱姆粘液×1 | 火把×1 | 3s | 默认 | 2 |
| 绳子 | 10003 | 杂草×3 | 绳子×1 | 4s | 默认 | 2 |
| 简易背包扩容 | 10004 | 皮革×2 + 绳子×1 | 背包扩容道具×1（临时+5格） | 10s | 采集Lv2 | 5 |
| 逃脱绳 | 10005 | 绳子×2 + 记忆粉尘×1 | 逃脱绳×1 | 8s | 地牢一层后 | 4 |

**注意**：逃脱绳占格子，可立即返回地牢入口。

### 2.2 锯木台（`Sawmill`）——对应 `WorkstationType.Sawmill`

锯木台用于将原木加工成木板、木方等建筑材料。素材有4级升级路线。

| 配方名称 | recipeID | 材料 | 产物 | 数量 | 制作时间 | 所需等级 | 解锁条件 | 经验 |
|----------|----------|------|------|------|----------|----------|----------|------|
| 木板 | 20001 | 木材×2 | 木板×1 | 5s | 制作Lv1 | 默认 | 3 |
| 木方 | 20002 | 木材×3 | 木方×1 | 8s | 制作Lv2 | 锯木台Lv2 | 5 |
| 木栅栏 | 20003 | 木板×2 | 栅栏×2 | 6s | 制作Lv1 | 默认 | 3 |
| 木支架 | 20004 | 木板×1 + 木棍×2 | 支架×1 | 7s | 制作Lv2 | 南瓜种植需求 | 4 |
| 压缩木板 | 20005 | 木材×5 | 压缩木板×1（高级建材） | 12s | 制作Lv4 | 锯木台Lv3 | 8 |
| 木齿轮 | 20006 | 木板×2 + 铁锭×1 | 齿轮×1 | 15s | 制作Lv5 | 自动进料器前置 | 10 |

**锯木台升级解锁**：
- Lv2（半自动）：解锁批量制作（5个），解锁木方配方
- Lv3（全自动）：解锁自动进料（需连接箱子），解锁压缩木板、齿轮

## 三、制作链条图（关键加工路径）

### 3.1 木材加工链
```
原木（砍树） → 锯木台 → 木板、木方
       ↳ 手工 → 木棍
木板 + 木棍 → 工作台 → 木工具、木栅栏
木板 + 铁锭 → 铁砧 → 铁质工具
```

### 3.2 矿石加工链
```
铜矿石 → 熔炉 → 铜锭 → 工作台 → 铜工具 / 铁砧 → 铜剑
铁矿石 → 熔炉 → 铁锭 → 工作台 → 铁工具 / 铁砧 → 铁剑 / 铁甲
铁矿石 + 煤炭 → 熔炉（高级）→ 钢锭 → 钢工具 / 钢剑
金矿石 → 熔炉 → 金锭 → 特殊配方
```

### 3.3 农业加工链
```
作物 → 烹饪台 → 料理（恢复精力/生命）
作物 → 种子机（需解锁）→ 种子
作物 + 杂草 → 堆肥桶（需解锁）→ 肥料
```

### 3.4 怪物掉落加工链
```
史莱姆粘液 + 木棍 → 火把
兽人獠牙 + 铁锭 → 钢剑（辅助材料）
记忆精华 + 水晶 → 记忆水晶 → 高级药水
```

### 3.5 自动化部件加工链
```
铁锭 + 齿轮 + 木板 → 手动洒水器
钢锭 + 齿轮 + 记忆水晶 → 自动洒水器
铁锭 + 齿轮 + 弹簧 → 自动进料器
```
