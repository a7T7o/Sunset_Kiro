# 篇章一：世界观与叙事设计

## 一、主角完整身世设定

### 核心设定：三重记忆封印

主角不是普通的穿越者，而是**主动封印自己记忆的“时空修补匠”**。他穿越的原因不是意外，而是**执行一项已经被自己遗忘的使命**。

**真名**：埃里克·“修补匠”·晨风（Erik “Tinker” Morningbreeze）
**年龄**：外表24岁，实际存在时间无法计算
**身份**：时空裂隙修补师——一个存在于世界夹缝中的古老职业，负责修复因时间线错乱而产生的“记忆裂隙”

**穿越前的事**：
埃里克原本生活在千年前的“辉煌纪元”——那个时代人类掌握着高度发达的魔导科技（对应游戏中的“高阶工作台”和“自动化”知识）。他发现了时间线正在发生严重的“记忆坍缩”：历史被篡改，无数人的存在正在被抹去。

为了阻止坍缩，他制造了**“记忆锚点装置”**（游戏中的关键剧情道具），将自己的所有知识压缩成“记忆碎片”，然后主动将自己投送到时间线的“创可贴点”——也就是现在这个破败村落所处的位置。这里曾经是辉煌纪元的一个科研前哨站，专门研究“记忆实体化”技术。

**为什么失忆**：
穿越过程中，记忆锚点装置受损，导致他的记忆被“打散”成碎片，散落在村落周围的时空裂隙中（即地牢深处）。他的失忆不是bug，而是**自我保护机制**——一次性恢复所有记忆会导致认知过载而精神崩溃。

**身世谜底逐步揭开**：
- **第一阶段觉醒**（村落复兴30%）：发现自己不是普通人，能看懂古代图纸
- **第二阶段觉醒**（村落复兴60%）：想起自己来自千年前的辉煌纪元
- **第三阶段觉醒**（村落复兴90%）：想起自己的使命——修复时间线，而完成使命的关键是**让这个村落恢复到当年的科技水平**，因为村落地下埋藏着当年实验室的核心装置

**最终真相**：
主角不是在“振兴一个陌生村落”，而是在**修复自己亲手创造的“记忆实验室”**。村落的每一个建筑、每一个工作台，都是他记忆的物理锚点。当最后一个工作台升级到最高级，当村落完全复兴，他的记忆将完整恢复，而玩家需要选择：
- 回到千年后继续当修补匠（离开这个时间线）
- 放弃记忆，留在这个村落当普通人（与NPC们一起生活）

这个选择是情感锚点的最终爆发。

## 二、村落详细设定

### 村落名称：灰碑谷 (Grey碑 Valley)

**地理定位**：
位于大陆最东端的“遗忘海岸”与“迷雾山脉”之间的峡谷地带。三面环山，一面临海，是一个天然封闭的地理单元。正因为封闭，才在历史长河中幸免于战乱，但也因此被世人遗忘。

**历史沿革（四阶段）**：

1. **辉煌纪元·前哨站时期（1000年前）**
   - 这里是帝国“记忆实体化研究所”第7号分站
   - 科学家们研究如何将人的记忆转化为可存储、可传输的物理实体
   - 村落中央的“灰碑”就是当年实验室的记忆核心塔——表面看是黑色石碑，实则是魔导装置

2. **大崩落时期（800年前）**
   - 实验失控，记忆能量爆发，导致时间线震荡
   - 研究所紧急关闭，大部分人员撤离
   - 少数选择留下的人，就是现在村民的祖先——他们负责看守遗址，等待“修补匠”的归来
   - 这个秘密代代相传，但经过几百年，已经变成了“等待一个能让我们重新过上好日子的旅人”的模糊传说

3. **沉寂时期（800年-现在）**
   - 由于记忆能量的残余辐射，这片土地变得“迟钝”——作物生长慢、动物繁殖难、矿石产量低
   - 年轻人陆续离开，只剩下走不动的老人和坚守祖训的几户人家
   - 与外界的联系彻底断绝，甚至地图上不再标注这个地点

4. **现在——主角到来时**
   - 全村只剩**12户人家，28口人**
   - 主要建筑：村长家（兼杂货铺）、铁匠铺（已停业3年）、废弃的锯木厂、摇摇欲坠的码头、一座破败的小教堂、一片荒废的农田
   - 村民靠打鱼和采集野果勉强维生，冬季经常有人饿死

### 村落核心建筑（必须基于现有美术资产设计）

| 建筑 | 对应美术资产 | 现状 | 复兴后 |
|------|--------------|------|--------|
| 村长家/杂货铺 | Tiny Wonder Farm - buildings | 二层木屋，一楼卖杂物 | 升级为“灰碑谷贸易站”，可交易稀有物品 |
| 铁匠铺 | Pixel Crawler - Buildings | 屋顶漏雨，风箱坏了 | 升级后解锁高级锻造配方 |
| 锯木厂 | Sawmill素材（4级升级路线） | 只有基础锯木台，手动操作 | 可升级到Lv3自动锯木厂 |
| 码头 | 需要自制（简单木栈道） | 只有一艘破渔船 | 修复后可钓鱼+出海贸易 |
| 小教堂 | Pixel Crawler - Buildings | 供奉“记忆女神”的废弃教堂 | 修复后可触发关键记忆闪回 |
| 灰碑（村落中央）| 需要自制（黑色石碑） | 布满苔藓的石碑 | 最终激活时发出光芒，解锁结局 |
| 农田区 | 已完整制作 | 3块小农田，种着勉强活着的作物 | 可开垦至30块农田 |
| 居住区 | Tiny Wonder Farm - buildings | 6间破旧小屋 | 修复后NPC搬回，好感度奖励 |

### 为什么被遗弃？——表层原因 vs 深层真相

**表层原因（村民以为的）**：
- 土地越来越贫瘠，作物收成差
- 鱼越来越少，海里有“不干净的东西”
- 年轻人去大城市就不回来了
- 这里“被神遗忘了”

**深层真相（玩家逐步发现的）**：
- 土地贫瘠是因为地下埋着记忆装置，它一直在“吸收”土地的生机来维持运转
- 海里的“不干净东西”其实是记忆能量的外溢形成的“记忆水母”（可做成稀有掉落）
- 不是被神遗忘，而是被“时间”遗忘——这个村落的时间流速比外界慢，导致外面过了800年，这里只感受到300年的流逝
- 灰碑一直在等待“修补匠”的回归，它发出的低频波动让村民产生“这里没希望”的潜意识，迫使村民离开——这是在保护他们，因为留在这里的人会被记忆能量影响，逐渐忘记外面的世界

**剧情反转点**：
当主角修复第一个工作台时，灰碑会发出微弱共鸣。老村长会突然流泪，说：“预言是真的……我们等的不是旅人，是修补匠。我爷爷的爷爷说过，会有人从天上掉下来，让灰碑重新发光。”

## 三、主线剧情完整脉络（4季节×28天结构）

### 整体结构：四幕剧，对应四个季节

#### 第一幕：春·萌芽（融入与信任）
**核心目标**：被收留 → 证明自己有用 → 获得村民信任 → 开启第一个工作台

| 时间节点 | 剧情事件 | 解锁内容 |
|----------|----------|----------|
| 第1天 | 主角坠落在灰碑下，被老村长女儿救起 | 解锁住处（废弃仓库） |
| 第3天 | 帮农夫老杰克修篱笆（需要砍5棵树） | 解锁基础斧头图纸 |
| 第5天 | 春耕开始，村民不够人手，主角帮忙种田 | 解锁锄头、种子袋 |
| 第7天 | 第一次记忆闪回：看到自己曾经站在这里指挥建造 | 解锁“锯木台”图纸（基础级） |
| 第10天 | 帮渔夫老汤姆修船（需要木材10） | 解锁码头、钓鱼竿 |
| 第14天 | 春季集市（迷你节日），村民凑钱买种子 | 解锁杂货店交易 |
| 第21天 | 发现村外洞穴有史莱姆骚扰羊圈 | 解锁基础武器（木剑） |
| 第28天 | 春末暴雨，码头被冲毁，需要重建 | 主线推进：需要更多木材→必须升级锯木台 |

**情感锚点**：
- 老村长女儿艾拉每天送饭，称呼从“旅人”变成“埃里克”（主角告诉她的名字）
- 孩子们开始跟着主角转，叫他“会修东西的大哥哥”

#### 第二幕：夏·危机（考验与抉择）
**核心目标**：应对夏季枯萎天 → 进入第一个地牢 → 解锁半自动

| 时间节点 | 剧情事件 | 解锁内容 |
|----------|----------|----------|
| 夏第3天 | 第一波枯萎天来袭，农田枯萎一半 | 解锁“水壶”应急，但效率低 |
| 夏第5天 | 艾拉病倒，需要“月光草”（只在洞穴深处生长） | 必须进入“灰碑地牢·第一层” |
| 夏第6天 | 地牢探索，击败骷髅守卫，找到月光草 | 第一次见到“记忆碎片”物品 |
| 夏第7天 | 喂药时接触艾拉，触发记忆闪回：想起自己曾经有个妹妹，长得像艾拉 | 解锁“制药台”基础 |
| 夏第10天 | 铁匠老乔治说：“如果有好矿石，我能重开铁匠铺” | 需要进入地牢二层挖矿 |
| 夏第14天 | 第二次枯萎天，这次更严重 | 必须升级水壶到铁质 |
| 夏第18天 | 地牢二层发现“铜矿脉”，但被锁着的门挡住 | 解锁“钥匙”系统 |
| 夏第20天 | 找到第一把钥匙，开门获得“古代熔炉图纸” | 解锁“熔炉”升级路线 |
| 夏第24天 | 用新熔炉炼出铜锭，升级铁匠铺 | 铁匠铺重开，解锁工具升级 |
| 夏第28天 | 艾拉康复，夏夜祭典，村民感谢主角 | 村长透露灰碑的秘密 |

**情感锚点**：
- 艾拉病危时的无力感 → 驱动玩家必须进地牢
- 铁匠重开店铺时的激动：“三十年！三十年没打铁了！”
- 夏夜祭典上，每个NPC都送主角一件小礼物

#### 第三幕·秋·收获与告别（成长与牺牲）
**核心目标**：应对不可逆枯萎 → 深入更深地牢 → 解锁自动化

| 时间节点 | 剧情事件 | 解锁内容 |
|----------|----------|----------|
| 秋第1天 | 发现部分作物即使浇水也在枯萎 | 引入“秋季枯萎不可逆”机制 |
| 秋第3天 | 老杰克说：“需要传说中的‘自动灌溉装置’” | 线索指向地牢三层 |
| 秋第5天 | 地牢三层发现“古代水管图纸”，但被精英怪守护 | 需要组队？不，只能自己上 |
| 秋第8天 | 击败精英骷髅战士，获得图纸 | 解锁“半自动灌溉”（需要手动灌水，自动洒水） |
| 秋第12天 | 安装灌溉系统，救下一半作物 | 村民信心恢复 |
| 秋第15天 | 发现地牢四层入口，但需要“三把不同钥匙” | 开启钥匙博弈核心阶段 |
| 秋第18天 | 收集钥匙过程中，触发大规模记忆闪回：想起自己是如何封印记忆的 | 解锁“高级工作台”图纸 |
| 秋第22天 | 进入四层，发现“记忆装置核心碎片” | 灰碑发光更明显 |
| 秋第25天 | 装置碎片激活，导致地牢怪物暴动 | 需要帮村民加固房屋 |
| 秋第28天 | 秋收节，收成比去年多三倍，但老杰克在庆祝中去世（老死） | 情感打击：复兴的代价 |

**情感锚点**：
- 老杰克去世前把祖传的“幸运锄头”送给主角
- 村民第一次说：“埃里克，你就是我们村的人了”
- 灰碑的光芒让村民开始相信“预言成真”

#### 第四幕·冬·真相与选择（觉醒与终结）
**核心目标**：应对寒冬 → 进入最深层地牢 → 恢复记忆 → 最终抉择

| 时间节点 | 剧情事件 | 解锁内容 |
|----------|----------|----------|
| 冬第1天 | 第一场雪，树木休眠，部分资源冻结 | 必须依靠存储过冬 |
| 冬第3天 | 发现储藏室粮食不够（需要更多自动生产） | 必须升级所有工作台到全自动 |
| 冬第5天 | 地牢五层，遇到第一个Boss：“记忆守卫·镜像” | 战斗系统考验 |
| 冬第8天 | 击败镜像，获得“完整的记忆核心” | 解锁最终工作台升级 |
| 冬第12天 | 安装核心，灰碑完全激活，整个村落被光芒笼罩 | 解锁所有隐藏配方 |
| 冬第15天 | 触发最长记忆闪回：看到自己1000年前站在这里，亲手启动装置 | 真相完全揭露 |
| 冬第18天 | 地牢六层（最终层）开启，Boss：“时间裂隙吞噬者” | 需要全村最好的装备 |
| 冬第22天 | 击败最终Boss，获得“时空锚点” | 可以选择修复时间线 |
| 冬第24天 | 艾拉发现主角不对劲：“你是不是要走了？” | 情感对话 |
| 冬第28天 | 冬临节前夜，村民为主角举办“感谢祭” | 最终抉择时刻 |

**最终抉择**：
- **选项A（离开）**：激活时空锚点，回到千年后继续当修补匠。离开前会看到村民送行的画面，艾拉含泪微笑。存档变成“修补匠的职责”，无法继续游玩，但获得特殊称号。
- **选项B（留下）**：将时空锚点埋入灰碑地下，永久封印时间裂隙。记忆会再次模糊，但保留所有技能。醒来时，艾拉在床边说：“你终于醒了，睡了两天，我还以为……欢迎回家。”游戏可以继续，没有结局，只有“和村民们一起生活”。

## 四、“记忆闪回”系统的叙事设计

### 触发机制（三种类型）

| 类型 | 触发条件 | 叙事表现 | 解锁内容 |
|------|----------|----------|----------|
| **工作台闪回** | 首次使用某类工作台累计10次 | 画面闪白，出现千年前自己使用同类工作台的虚影 | 解锁该工作台下一级升级图纸 |
| **遗物闪回** | 在地牢拾取特定“记忆碎片”物品（不可堆叠） | 碎片融入身体，看到一段完整记忆画面 | 解锁隐藏配方/技能 |
| **地点闪回** | 站在特定坐标（灰碑下/实验室遗址/码头尽头） | 听到回音式对话：“你确定要这么做吗？”“我确定。” | 解锁相关剧情线索 |

### 记忆碎片物品设计（对应KeyLockData框架）

**利用 KeyLockData 的子类化潜力**：
已有 `KeyLockData` 可以继承扩展为 `MemoryShardData`，保留 `unlockChance` 字段改为“记忆完整度概率”。

| 记忆碎片名称 | ID范围 | 获取方式 | 记忆内容 | 解锁效果 |
|--------------|--------|----------|----------|----------|
| 碎片·起始 | 1450 | 地牢一层固定宝箱 | 自己在实验室启动装置 | 解锁锯木台Lv2 |
| 碎片·亲情 | 1451 | 艾拉好感度3心赠送 | 妹妹的生日，自己在做木马 | 解锁“玩具”配方（可送NPC加好感） |
| 碎片·责任 | 1452 | 击败骷髅战士掉落 | 师父临终托付：“修补匠的职责” | 解锁所有工具铁质升级 |
| 碎片·恐惧 | 1453 | 地牢三层隐藏房间 | 看到时间裂隙吞噬同伴 | 解锁“逃脱绳”配方 |
| 碎片·希望 | 1454 | 夏季祭典后自动获得 | 村民们围着自己笑 | 解锁“祝福”buff配方 |
| 碎片·真相 | 1455 | 最终Boss前 | 自己就是制造裂隙的人 | 解锁最终武器图纸 |

### 记忆闪回的UI/UX表现

利用现有 `SimpleEventData` 框架：
- `eventType = ShowMessage` + `displayImage`（记忆画面）
- `displayDuration` 设置为5-8秒
- 同时播放特定音效（模糊的人声+心跳声）
- 闪回期间玩家无法移动（但可以按任意键跳过）

**叙事原则**：
- 前期闪回：模糊、碎片化、只有画面没有声音
- 中期闪回：画面清晰，开始出现关键词
- 后期闪回：完整对话，情感冲击

## 五、NPC角色设定（6个核心NPC + 2个隐藏）

### NPC设计框架（基于现有素材）

所有NPC必须满足：
- 有可识别的美术素材（来自Pixel Crawler NPC或待整合角色包）
- 有日常行为路径（需要新增，但策划案中描述即可）
- 有好感度任务线（至少3个节点）
- 与村落复兴的某个系统绑定

---

#### NPC 001：艾拉·河风 —— 治愈系女主角

**素材来源**：Pixel Crawler - Npc's - Rogue（女性盗贼造型，调整颜色）
**年龄**：22岁
**职业**：村长的女儿，兼职护士/杂货店员
**性格**：温柔坚韧，爱笑，但眼底有淡淡的忧郁（母亲在她5岁时出海捕鱼未归）
**日常行为**：
- 上午：在杂货店帮忙
- 下午：去海边眺望（等母亲）
