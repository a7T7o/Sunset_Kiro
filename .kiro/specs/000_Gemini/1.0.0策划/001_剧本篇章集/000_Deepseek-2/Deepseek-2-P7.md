# 篇章七：UI/UX设计指南

## 一、整体设计风格

### 1.1 视觉基调
- **主风格**：手绘书本风格 + 星露谷式像素风。所有面板采用羊皮纸背景，边框装饰有藤蔓、木质或铁艺花纹。
- **字体**：使用像素风格字体（如Press Start 2P或自定义像素字体），标题略带阴影，正文清晰易读。
- **配色**：温暖的大地色系为主（棕、黄褐、暗红），辅以淡绿、淡蓝表示状态。重要信息用金色或红色高亮。
- **素材来源**：`Complete_UI_Book_Styles_Pack_Free` 提供面板边框、标签页、按钮基础；`Stardew Valley` 风格作为布局参考；结合游戏内已有图标（`Icons/Resources.aseprite` 等）。

### 1.2 交互原则
- **一致性**：所有界面采用相似的布局结构（标题栏、关闭按钮、内容区、标签页）。
- **反馈即时性**：点击按钮有按下效果，拖动物品有阴影跟随，操作成功/失败有短暂提示（文字或音效）。
- **快捷键支持**：常用界面（背包、制作、地图）可通过快捷键一键打开/关闭。
- **暂停机制**：打开任何UI面板时，游戏时间暂停（但动画可继续），保证玩家有充足时间操作。

## 二、各功能界面详细设计

### 2.1 主界面（游戏运行时的HUD）

**布局草图**：
- **屏幕左上角**：日期/季节/天气信息。显示“春1日 星期一 晴天”字样，右侧小图标表示天气（太阳、雨滴、雪花）。
- **屏幕右上角**：时间/精力/金钱条。
  - 时间：钟表图标 + 当前时间（如“6:00 AM”）。
  - 精力条：绿色进度条，右侧显示当前精力/最大精力（如“120/200”），以心形图标装饰。
  - 金钱：金币图标 + 当前持有金额。
- **屏幕底部中央**：快捷栏（10格）。每格显示物品图标和数量（若可堆叠）。当前选中格高亮（白色边框）。左右有箭头指示可翻页（若有更多快捷栏）。
- **屏幕右下角**：小地图缩略图（点击可放大地图）。显示玩家位置（绿点）、重要建筑（房屋图标）、地牢入口（骷髅图标）。
- **屏幕左下角**：任务追踪区（显示当前主线/支线任务目标，可折叠）。
- **屏幕中央上方**：偶尔出现提示信息（如“按E打开背包”），淡入淡出。

**交互**：
- 鼠标悬停在物品上显示名称和品质颜色。
- 点击快捷栏物品可选中，再次点击使用/装备。
- 数字键1-10直接切换快捷栏。

### 2.2 背包界面

**布局草图**：
- **背景**：羊皮纸卷轴，边缘有铁质包角，上方有系带装饰。
- **标题栏**：左侧“背包”字样，右侧关闭按钮（X）。
- **主内容区**：物品网格（初始6×2=12格，可通过扩容增加至8×6=48格）。每个格子有深色背景，边框略凸起。格子上显示物品图标、堆叠数量（若大于1）。
- **右侧信息栏**：
  - 玩家形象剪影（可显示当前装备外观）。
  - 装备槽位：头盔、衣服、裤子、鞋子、戒指×2（图标表示，可拖拽装备）。
  - 统计信息：防御力、攻击力、暴击率等（根据装备计算）。
- **底部工具栏**：
  - “整理”按钮（调用 `InventorySortService` 自动排序）。
  - “丢弃”按钮（选中物品后点击丢弃，需二次确认）。
  - “拆分”按钮（选中可堆叠物品后，弹出滑块选择拆分数量）。
- **标签页**（如果有分类）：可通过左右箭头切换“全部”、“工具”、“材料”、“食物”等（利用 `ItemData.category`）。

**交互**：
- 左键点击物品拿起，再点击空位或其他物品放置（若可堆叠则合并）。
- 右键点击物品快速使用/装备。
- 拖拽物品到装备槽位自动装备。
- 拖拽物品到界面外可丢弃（需确认）。
- 鼠标悬停在物品上显示详细属性（如武器攻击力、食物恢复值）。

### 2.3 箱子界面

**布局草图**：
- 整体分为左右两栏：
  - **左侧**：玩家背包（与背包界面相同布局，但标题改为“背包”）。
  - **右侧**：箱子存储区（根据箱子容量显示对应格数：12/24/36/48格，使用 `Box_12/24/36/48` UI预制体）。
- **中间**：有箭头图标表示物品可双向移动，也可直接拖拽。
- **顶部**：箱子名称（如“木箱”），若有锁则显示锁图标，并显示开锁概率（如果玩家持有钥匙，鼠标悬停显示成功率）。
- **底部**：“整理箱子”按钮（仅整理箱子侧）、“全部取回”按钮（将箱子所有物品移至背包，需确认）。

**交互**：
- 拖拽物品在两栏之间移动。
- Shift+左键点击物品快速移动至另一侧。
- 右键点击物品快速使用（若在背包侧）或取出（若在箱子侧）。
- 若箱子上锁，点击箱子侧任意格子时弹出钥匙选择界面（列出背包中可用钥匙），选择后执行开锁尝试，成功则解锁并打开箱子，失败钥匙消失。

### 2.4 制作界面

**布局草图**：
- 背景：类似工作台图纸，有齿轮、锤子装饰。
- **左侧**：分类标签（手工、烹饪、冶炼、锻造、制药、锯木等），每个标签有对应图标。
- **中间**：配方列表。每个配方显示：
  - 产物图标、名称、所需材料列表（用小图标+数量表示）。
  - 若玩家材料足够，背景高亮；若不足，材料数量变红。
  - 右侧显示“制作”按钮（可制作1个）和“批量”按钮（若工作台支持）。
- **右侧**：当前工作台信息（如果站在工作台前）。显示工作台名称、等级、剩余燃料（若有）、自动化状态（Lv3显示“自动生产中”）。
- **底部**：玩家背包材料统计？或简单显示当前选中配方的材料需求与持有量对比。

**交互**：
- 点击配方选中，再点击制作按钮制作。
- 双击配方直接制作（若材料足够）。
- 批量制作时弹出滑块选择数量（1-最大可制作数）。
- 若站在工作台前，制作界面自动切换到对应工作台的配方分类。

### 2.5 技能界面

**布局草图**：
- 背景：展开的书本，左右页分别显示不同内容。
- **左页**：技能列表（战斗、采集、制作、烹饪、钓鱼）。每个技能显示：
  - 技能图标（剑、斧、锤、锅、鱼）。
  - 当前等级（Lv.X）和进度条（经验/下一级所需经验）。
  - 等级奖励简述（如“Lv2：树木掉落+10%”）。
- **右页**：当前选中技能的详细加成列表，以及已解锁的配方/能力。
- **底部**：总等级（所有技能等级之和）和称号（如“新手”、“工匠”）。

**交互**：
- 点击技能图标切换右页内容。
- 鼠标悬停在进度条上显示具体经验数值。

### 2.6 好感度/社交界面

**布局草图**：
- 背景：心形装饰的羊皮纸。
- **左侧**：村落NPC头像列表（圆形头像，使用NPC素材裁剪）。每个头像右侧显示心形数量（0-10心），名字。
- **右侧**：选中NPC的详细信息：
  - 大尺寸头像。
  - 好感度进度条（当前值/下一心所需）。
  - 个人简介（性格、背景简述）。
  - 礼物偏好列表（喜爱、喜欢、讨厌），用小图标表示。
  - 任务状态（当前进行中的任务，点击可追踪）。
- **底部**：“送礼”按钮（若在NPC附近，可直接送礼）、“对话记录”按钮。

**交互**：
- 点击左侧NPC切换右侧信息。
- 好感度进度条鼠标悬停显示具体数值。

### 2.7 地图界面

**布局草图**：
- 背景：手绘风格的地图纸，边缘有烧焦痕迹。
- **主区域**：村落及周边地图。使用 Tilemap 渲染的缩略图。标注：
  - 玩家位置（闪烁绿点）。
  - 重要建筑（图标+名称，如“杂货店”、“铁匠铺”）。
  - 地牢入口（骷髅图标）。
  - 资源点（树木、矿石等小图标）。
  - 当前任务目标位置（黄色感叹号）。
- **右侧信息栏**：
  - 当前区域名称。
  - 天气/季节信息。
  - 可传送点列表（若已解锁传送阵）。
- **底部**：缩放滑块（+/-）和“重置”按钮（恢复默认缩放）。

**交互**：
- 鼠标滚轮缩放地图。
- 点击地图上的标记可查看名称，若有传送功能则点击传送点可快速旅行（需消耗时间或物品）。

### 2.8 任务/日志界面

**布局草图**：
- 背景：带锁扣的日志本。
- **左侧**：任务分类标签（主线、支线、已完成）。
- **中间**：任务列表。每个任务显示：
  - 任务名称。
  - 当前目标（简短描述）。
  - 任务进度（如“收集木材 5/10”）。
  - 奖励预览（经验、金钱、物品图标）。
- **右侧**：选中任务的详细描述（背景故事、对话摘录、后续目标）。
- **底部**：“追踪”按钮（将任务显示在主界面任务追踪区）。

**交互**：
- 点击任务列表项切换右侧详情。
- 已完成任务显示为灰色，可查看回顾。

### 2.9 设置/选项界面

**布局草图**：
- 背景：机械风格的金属板（但保持整体风格统一）。
- **标签页**：游戏设置、图像设置、声音设置、控制设置。
- **游戏设置**：
  - 时间流速滑块（0.5x-2x）。
  - 自动保存间隔（5/10/15分钟）。
  - 显示伤害数字（开/关）。
- **图像设置**：
  - 分辨率下拉框。
  - 全屏/窗口模式。
  - 像素风格滤镜（如CRT、无）。
- **声音设置**：
  - 主音量、音乐音量、音效音量滑块。
- **控制设置**：
  - 按键绑定列表（可点击后按下新键重新映射）。
- **底部**：“恢复默认”、“应用”、“关闭”按钮。

**交互**：
- 滑块拖动实时预览效果。
- 按键绑定时检测冲突并提示。

### 2.10 对话/剧情界面

**布局草图**：
- 底部弹出对话框（半透明羊皮纸背景，边缘卷曲）。
- 左侧显示说话者头像（NPC头像或玩家剪影）。
- 右侧显示对话文本（每行显示完逐字出现，按空格/回车快速显示）。
- 底部显示选项（若有分支），用数字1-4标记，可鼠标点击或按数字键选择。
- 右上角有时显示任务更新提示（淡入淡出）。

**交互**：
- 点击对话框任意区域或按空格继续对话。
- 选择选项后剧情分支推进。

### 2.11 记忆闪回界面

**布局草图**：
- 全屏半透明黑色遮罩，中间出现记忆碎片旋转效果。
- 画面中央显示闪回图像（静态图，利用 `SimpleEventData.displayImage`），四周有闪烁的光芒。
- 底部显示字幕（回忆对话），字体略带模糊，有回音效果。
- 进度条表示闪回持续时间（或可点击跳过）。

**交互**：
- 按空格键跳过闪回（但会错过剧情）。
- 闪回期间无法操作角色。

## 三、交互流程设计

### 3.1 打开/关闭界面
- **背包**：按B键或点击快捷栏上方背包图标打开。
- **制作**：站在工作台前按E键，或按C键打开手工制作界面（若无工作台则显示手工配方）。
- **地图**：按M键打开。
- **任务**：按J键或L键打开。
- **技能**：按K键打开。
- **社交**：按N键打开（或点击HUD上的NPC图标）。
- **设置**：按ESC键打开主菜单，选择“设置”。
- **关闭**：按ESC关闭当前界面（若有多个界面叠放，逐层关闭），或点击右上角X按钮。

### 3.2 物品操作
- **拾取**：靠近地面物品，按F键或鼠标左键点击拾取（自动放入背包）。
- **使用**：选中快捷栏物品，点击左键使用（工具）或右键食用（食物）。
- **丢弃**：打开背包，拖动物品到界面外，弹出确认框“确定丢弃XX？”选择“是”则物品消失（生成 `WorldItemDrop` 在地面）。
- **拆分**：按住Shift键点击可堆叠物品，弹出拆分滑块。

### 3.3 交互反馈
- 操作成功：轻微音效（如纸张翻动、金币声）。
- 操作失败：红色文字提示（如“材料不足”、“精力不够”、“背包已满”）并播放错误音效。
- 任务更新：右上角弹出绿色提示框，显示任务名称和状态。

### 3.4 与NPC交互
- 靠近NPC，头顶出现对话气泡（...），按E键对话。
- 对话中触发剧情或打开商店/任务界面。
- 送礼：在背包中选中物品，靠近NPC，点击NPC身上出现的礼物图标即可赠送（若NPC喜欢则播放心动画）。

## 四、特殊界面设计

### 4.1 自动化工作台界面（Lv3）
- 当工作台升级到Lv3并连接输入/输出箱后，打开工作台界面时新增“自动生产”标签页。
- 布局：左侧显示配方列表，右侧显示“输入箱”和“输出箱”的当前内容预览（只显示前几个物品和总数量）。
- 底部有“启动自动生产”按钮，点击后设置配方和数量（无限/指定数量），工作台开始自动工作。
- 界面上有开关按钮“停止生产”。

### 4.2 地牢死亡全掉落提示
- 当玩家在地牢死亡时，弹出红色警告界面：
  - 背景：骷髅头+交叉骨。
  - 文字：“你倒在了地牢中……所有物品遗落在原地。你会在家中醒来，精力减半。”
  - 按钮：“确认”（返回游戏）。
- 同时自动记录死亡位置，生成墓碑标记（可返回拾取）。

## 五、UI适配与分辨率
- 所有UI使用Canvas Scaler设置为“Scale with Screen Size”，参考分辨率1920×1080，以像素完美缩放。
- 支持16:9和16:10比例，两侧留黑边或背景延伸。
